This repository contains two beginner-friendly Python programs:

Task 1: Perform Basic Mathematical Operations

Task 2: Create a Personalized Greeting

Task 1 takes input of two numbers from the users and performs simple mathematical tasks like addition , subtraction , multiplication and division
Task 2 takes user's frist name and last name and prints a greeting sign which says " Hello" , first and last name of the user,"!" "Welcome to the python programming")

evenorodd.py is a python program that takes a numeric value as input from the user and checks whether the number is odd or even and prints the result accordingly.It uses a if-else statement for checking if the number is odd or even.

sum1to50 is a python program that prints the sum from 1 to 50. It uses a for loop for iteration.
